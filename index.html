Abstract: CIFNet-Pest introduces a hybrid convolution, involution, and Fourier network model to classify plant pests with high feature learning adaptability. The architecture extracts local patterns using VGG16, refines coordinates-dependent responses using involution kernels, and models global contexts via Fourier mixing (FNet). It achieves high recognition rates: 95.99% accuracy on jute pests and 94.48% on tomato pests, using only 10.07 million parameters, proving its viability for deployment in edge systems.
